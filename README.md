# AstroSounds

## Purpose
A website that provides Spotify playlists based on your daily Horoscope results

## Built With
* HTML 
* CSS
* Materialize
* Javascript
* JQuery 
* Aztro API 
* TextProbe API
* Spotify Scraper API
